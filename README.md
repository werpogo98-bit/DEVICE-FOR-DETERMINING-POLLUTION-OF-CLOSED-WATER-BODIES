В данном репозитории представлена автономная версия ПО для устройства.

Интервал замеров: 45 минут.

Энергосбережение: Использован режим Deep Sleep.

Хранение данных: Энергонезависимая файловая система LittleFS.

Эта версия предназначена для длительной работы в полевых условиях, в отличие от демонстрационной версии (с интервалом 10 сек), использованной на защите.

Так же представлен Python скрипт для извлечения показаний с датчика в базу данных
